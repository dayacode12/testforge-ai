#  TestForge AI

##  Overview
TestForge AI is an AI-powered test automation tool that generates structured test cases from plain English feature descriptions.

This project demonstrates how modern SDETs can leverage Large Language Models (LLMs) to enhance automation workflows and reduce manual effort.

---

##  Features
- Generate test cases using AI
- API-based architecture using FastAPI
- Modular and scalable backend design
- Clean separation of concerns (routes, services, models)
- Easy integration with automation frameworks

---

##  Architecture

User Input → FastAPI → AI Engine → Formatter → JSON Response

---

## Tech Stack
- Python 
- FastAPI 
- OpenAI API 
- Pydantic 
- Uvicorn
