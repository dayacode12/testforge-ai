## TROUBLESHOOTING

### If "docker: command not found"
- Jenkins container needs Docker CLI
- The containers should still communicate
- If stuck, restart Jenkins: `docker restart jenkins`

### If "Connection refused to ollama:11434"
- Check network: `docker network inspect shared-network`
- Make sure all 3 containers are connected
- Verify: `docker ps` shows all three running

### If build shows "cannot find Jenkinsfile"
- Make sure your changes are committed and pushed to GitHub
- The Jenkinsfile must be in the root of the repo

### If "Git connection timeout"
- Check internet connection
- Wait a bit and retry
